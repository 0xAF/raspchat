# A basic SocketIO chat server in Go

# Demo

Basic demo is available [Here](http://raspchat.ngrok.io). Do note the server is running on Raspberry Pi 1 Model B.

## Features:

 * Basic Gif support
 * Basic nick support
 * Channel support
 * Markdown support
